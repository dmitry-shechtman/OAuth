Welcome to Portable OAuth

What

OAuth 1.0 Consumer .NET Portable Class Library supporting:

  • .NET Framework 4.5
  • Windows Store Apps
  • Xamarin.iOS
  • Xamarin.Android
  • Portable Class Libraries
Why

I needed a PCL OAuth consumer for my E*TRADE .NET Portable Client Library, and I couldn't find one that just worked.

Thanks
  • Yoshifumi Kawai for AsyncOAuth that got me started,
  • The Legion of the Bouncy Castle for the crypto, and
  • Oren Novotny for Bouncy Castle-PCL, which (hopefully) makes this somewhat better than AsyncOAuth.
